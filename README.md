## A propos

Le BOT à pour but de crée une radio sur discord pour envoyé des messages radio reçus par le destinataire en message privées et stocker dans un channel accessible uniquement par le MJ
  

Idées d'utilisations:

* Pour des parties de JDR de Warhammer

* Pour des serveurs FiveM
 

## Prérequis

  

Le bot à besoin d'être sur un serveur NODE, soit en local voir le petit guide : 
Ou bien un hébergeur 

  

### Prérequis Discord

  

Pour les prérequis discord, il vous faut crée un bot [ici](https://discordapp.com/developers/applications/) et mettre le token du bot dans idbot:

ensuite dans channelmj: vous mettez l'id du channel ou seul le MJ a accès (channel de log)

### Installation

  

1. Sur votre serveur node ou local, mettez vous dans le dossier bot et faite

```sh
npm install
```

2. Lancer le BOT

```sh
node index.js
```

## Utilisation

Les commandes du bot sont simples en fonction du préfixe voulut

- ping : Pour tester si le bot est bien lancé
- help : Pour avoirs le format de commande pour envoyé un message
- radio : radio _ @nomdestinataire _ date _ Message a ecrire (N'oubliez pas les espaces entres les _)
